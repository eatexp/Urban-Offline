# Urban-Offline

## Overview
Urban-Offline is an Offline-First Emergency Preparedness Application designed to provide critical information and tools when internet connectivity is unavailable.

## Features
- **Offline Access**: Core functionality works without an internet connection.
- **Emergency Guides**: Access to first aid and survival guides.
- **Resource Maps**: Locate offline resources (shelters, water, etc.).

## Tech Stack
- **Vite**: Fast build tool and dev server.
- **React**: UI library.
- **VitePWA**: Progressive Web App support.
- **Vanilla CSS**: Custom styling.

## Getting Started

### Prerequisites
- Node.js (LTS)

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
Build for production:
```bash
npm run build
```
